Capstone Project – ShiftLite

Project Overview

ShiftLite is a capstone project focused on improving staff coordination, scheduling, and communication within a workplace environment. The system is designed to support features such as role-based access, announcements, and task or inventory tracking.

My Role

As part of a team, I contributed to feature planning, backend logic design, and collaborative development using GitHub. I participated in sprint planning, implementation, and iterative improvements throughout the project lifecycle.

Key Features

  • User authentication and role-based access
  • Staff scheduling and coordination
  • Announcements and internal communication
  • Task and inventory management

Technologies Used

  • Full-stack web technologies
  • API development
  • Database integration
  • GitHub for version control and team collaboration
